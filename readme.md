## Building
Build with
```bash
docker-compose build
```

## Starting
Run with
```bash
docker-compose up
```

## Using

Exposes TYPO3 instance on ports 80 and 443 by default  
Database host is `db`  
Database user, name, password and root password are `dev`  

Go to `http://<host_ip>/typo3/install` to launch the setup
